API Utilitários

Serviços práticos para desenvolvedores

Online

Dados e Segurança

  • GET /api/perfis

    Retorna perfis de usuários com informações básicas

  • GET /api/generate-password?length=12&special=true

    Gera senhas seguras com opções de personalização

  • GET /api/my-ip

    Retorna seu endereço IP público e informações geolocalização

Documentos e Pagamentos

  • GET /api/cnpj?cnpj=00000000000191

    Consulta de CNPJ com dados da Receita Federal

  • GET /api/pix?chave=12345678900&valor=100.00

    Geração de QR Code PIX para pagamentos

Como usar a API

Todos os endpoints respondem a requisições GET. Para endpoints que requerem parâmetros, inclua-os na query string da URL. As respostas são retornadas no formato JSON.
